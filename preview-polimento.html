center_focus_strong MELHORIA 5
Auto-Focus em Modais
Ao abrir qualquer modal, o primeiro campo de input recebe foco automaticamente. O cursor já fica piscando no campo — usuário começa a digitar sem precisar clicar.
Teste — abra o modal e observe o cursor

Clique no botão para abrir o modal. O campo "Nome do Fornecedor" já aparece com o cursor piscando — sem precisar clicar nele.

check COMPORTAMENTO ESPERADO
  • Modal abre
  • Primeiro input recebe focus() automaticamente
  • Cursor pisca no campo
  • Usuário digita direto
Código adicionado no infra.js (função showModal)
function showModal(id) {
  const el = document.getElementById(id);
  if (el) {
    el.classList.add('active');
    // NOVO: foca no primeiro input do modal
    setTimeout(() => {
      const firstInput = el.querySelector('input, select, textarea');
      if (firstInput) firstInput.focus();
    }, 80);
  }
}
