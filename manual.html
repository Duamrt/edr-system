Como fazer login

Acesse sistema.edreng.com.br no navegador
Digite seu usuario e senha
Clique em ENTRAR ou pressione Enter
No computador, use as teclas F1 a F12 para navegar rapidamente entre os menus.
No celular, use o menu inferior. Toque em MAIS para ver todos os modulos.
📊 Resumo (Dashboard) F1 ▼
Admin Operacional

Visao Admin

Painel completo com dados financeiros de todas as obras:

  • Painel financeiro — receita total, custo total e saldo
  • Grafico mensal — evolucao de entradas mes a mes
  • Detalhe por obra — custos separados por etapa
  • Comparativo — variacao mes anterior vs atual

Visao Operacional

Dashboard simplificado com foco na operacao:

  • Obras ativas — cards com obras em andamento
  • Estoque — top 8 materiais disponiveis
  • Ultimos lancamentos — 8 mais recentes
  • Botoes rapidos — atalho para Lancar NF e Estoque
🏗 Obras F2 ▼
Admin Operacional

Gerenciamento completo das obras/projetos da empresa.

Visualizar obras

Acesse OBRAS (F2) — as obras ativas aparecem em cards
Clique em um card para abrir o detalhe da obra
Na aba Materiais, veja todos os lancamentos separados por etapa (centro de custo)
Na aba Adicionais, veja servicos extras contratados

Criar nova obra

Clique no botao + NOVA OBRA
Preencha: nome da obra, proprietario, endereco
Clique em SALVAR

Arquivar obra concluida

Abra o detalhe da obra
Clique em CONCLUIR OBRA
O sistema gera automaticamente o Termo de Entrega (Kit Digital)
A obra vai para a secao de arquivadas
Para ver obras arquivadas, ative o botao Mostrar Arquivadas na tela de obras.

Gerar Termo de Entrega

No detalhe da obra, clique em GERAR TERMO
O sistema abre uma pagina com o termo pronto para impressao/PDF
Use Ctrl+P para salvar como PDF ou imprimir

Filtrar lancamentos

Dentro do detalhe da obra, use o filtro por categoria ou por data
Clique na categoria para expandir/recolher os itens
📦 Estoque F3 ▼
Admin Operacional

Controle completo de materiais no almoxarifado. O saldo e calculado automaticamente:

Saldo = (Notas Fiscais + Entradas Diretas + Ajustes) − Distribuicoes para obras

Consultar saldo

Acesse ESTOQUE (F3)
Os materiais aparecem em cards com o saldo atual
Use a barra de busca para filtrar por nome do material
Cards com saldo zerado ou negativo ficam sinalizados com cores diferentes.

Entrada Direta (material sem NF)

Para registrar material que chegou sem nota fiscal (fiado, doacao, sobra de outra obra):

Clique no botao 📥 ENTRADA (barra superior)
Escolha o destino: Estoque ou Direto na Obra
Preencha: material, quantidade, unidade, preco, fornecedor
Clique em SALVAR
Estoque = material fica guardado, voce distribui depois. Obra = material ja vai direto pro custo da obra, sem passar pelo estoque.

Saida de material (distribuir para obra)

Para enviar material do estoque para uma obra:

Clique no botao 📤 SAIDA na barra superior, ou clique no botao 📤 do proprio card do material
Selecione a obra destino
Selecione a etapa (centro de custo: alvenaria, eletrica, etc.)
Informe a quantidade
Clique em CONFIRMAR
O sistema usa logica FIFO (primeiro que entrou, primeiro que sai). O custo e calculado automaticamente pelo preco de entrada e o lancamento e criado na obra.

Ajuste de estoque (somente Admin)

Somente Admin

Para corrigir diferencas entre o sistema e a realidade:

Clique no botao 📋 AJUSTE (barra superior)
Busque o material pelo nome
O sistema mostra o saldo atual
Escolha o tipo:
  • Inventario — material que ja existia (sempre positivo)
  • Contagem Fisica — diferenca da contagem real (+ ou −)
  • Correcao — erro, perda ou extravio (+ ou −)
Informe a quantidade e o motivo
Clique em SALVAR
Ajustes nao geram custo em obra. Servem apenas para corrigir o saldo do estoque.
🧾 Historico de Notas Fiscais F4 ▼
Admin Operacional

Lista completa de todas as notas fiscais lancadas no sistema.

Consultar notas

Acesse HISTORICO (F4)
As notas aparecem em ordem cronologica (mais recente primeiro)
Clique em uma nota para ver os itens detalhados

Informacoes de cada nota

  • Numero da NF, data, fornecedor, CNPJ
  • Valor bruto, frete, imposto
  • Itens — descricao, quantidade, unidade, preco, total
  • Credito — se gera credito fiscal ou nao
  • Obra destino — pra onde foi o material
➕ Lancar Nota Fiscal F5 ▼
Admin Operacional

Formulario para registrar novas notas fiscais no sistema.

Lancamento manual

Acesse LANCAR NF (F5)
Preencha o cabecalho: data, numero da NF, fornecedor, CNPJ
Selecione a obra destino:
  • EDR = material vai para o estoque
  • Nome da obra = custo vai direto para a obra
Adicione os itens: descricao, quantidade, unidade, preco unitario
Marque se o item gera credito fiscal ou nao
Preencha frete e imposto se houver
Clique em SALVAR NOTA
O sistema sugere materiais do catalogo conforme voce digita. Isso padroniza os nomes e categoriza automaticamente.

Importacao Rapida (copiar/colar)

Para notas com muitos itens, voce pode colar o texto direto:

Clique em IMPORTAR
Cole o texto da nota (formato: descricao, qtd, preco — separados por tab, ponto-e-virgula ou pipe)
O sistema interpreta automaticamente e mostra um preview
Revise, ajuste se necessario, e clique em CONFIRMAR

Importacao XML (NF-e)

Clique em IMPORTAR XML
Selecione o arquivo .xml da nota fiscal eletronica
O sistema extrai automaticamente: fornecedor, CNPJ, itens, valores
Revise e confirme
Confira sempre a obra destino antes de salvar. Material com obra = EDR vai pro estoque. Qualquer outra obra vai direto pro custo.
📖 Guia de Creditos NF F6 ▼
Admin Operacional

Tabela de referencia para saber quais materiais geram credito fiscal na construcao civil.

Como usar

Acesse GUIA NF (F6)
Consulte a tabela: itens com ✓ geram credito, itens com ✗ nao geram

Exemplos

Gera Credito ✓Nao Gera ✗
Cimento, ferro, madeiraCombustivel
Argamassa, tinta, pisoAlimentacao
Telhas, portas, janelasLimpeza, ferramentas
Tubos, fios, concretoMobiliario, multas
Frete de materiaisEPI (em geral)
Ao lancar uma NF, o sistema tenta classificar automaticamente o credito baseado no catalogo.
👷 Diarias F7 ▼
Admin Operacional Mestre

Controle de diarias da equipe de obra. O mestre de obras tem acesso exclusivo a este modulo.

Entendendo a estrutura

  • Quinzena — periodo de pagamento (ex: 01/03 a 15/03)
  • Periodos — manha, tarde, noite (cada um conta como 0,5 ou 1 diaria)
  • Extras — bonus ou descontos por quinzena

Lancar diarias do dia

Acesse DIARIAS (F7) — a aba Registro ja vem selecionada
No campo de texto, digite no formato: nome quantidade
  • Exemplo: anderson 3 = Anderson trabalhou 3 periodos
  • Exemplo: nego 2,5 = Nego trabalhou 2,5 periodos
  • Funciona com apelidos (nego, val, zezao, etc.)
O sistema interpreta e mostra o resumo
Confirme clicando em LANCAR
Voce pode lancar varios funcionarios de uma vez, um por linha.

Ver calendario de diarias

Na aba Registro, role para baixo
Os dias aparecem em ordem, clique para expandir e ver quem trabalhou
Para excluir um dia inteiro, clique no botao de excluir do dia

Ver folha de pagamento

Clique na aba Folha
A folha mostra: funcionario, dias trabalhados, valor da diaria, total, extras, valor final
Clique em EXPORTAR PDF para salvar/imprimir

Adicionar extras (bonus/desconto)

Clique na aba Extras
Clique em + EXTRA
Selecione o funcionario, informe o valor (positivo = bonus, negativo = desconto) e a descricao
Clique em SALVAR

Gerenciar quinzenas

No topo do modulo, veja a quinzena ativa
Para criar uma nova, clique em NOVA QUINZENA — o sistema sugere as datas automaticamente
Para trocar de quinzena, use o seletor no topo

Gerenciar equipe (Admin)

Somente Admin
Clique em EQUIPE
Para adicionar: preencha nome, cargo, valor da diaria, apelidos
Para desativar: clique no botao ao lado do nome (ex: funcionario que saiu)
Para editar valor: clique no nome e altere

Lancar diarias como custo na obra (Admin)

Somente Admin
Clique em LANCAR EDR
Selecione a obra
O sistema calcula o custo total de diarias da quinzena
Confirme para gerar o lancamento na obra
📂 Catalogo de Materiais F8 ▼
Admin Operacional

Base de materiais padronizados. Quando voce lanca uma NF, o sistema usa o catalogo para sugerir nomes e categorizar automaticamente.

Consultar materiais

Acesse CATALOGO (F8)
Use a busca ou filtre por categoria
Cada material mostra: codigo, nome, unidade, categoria, preco medio

Cadastrar novo material

Clique em + NOVO MATERIAL
Preencha: nome, unidade (UN, m2, kg, etc.), categoria
Clique em SALVAR — o codigo e gerado automaticamente
Materiais tambem sao cadastrados automaticamente quando voce lanca uma NF com itens novos. Eles aparecem com um badge de "auto-cadastrado" para voce revisar.

Reconciliacao (Admin)

Somente Admin

Ferramenta para vincular itens orfaos (lancados sem match no catalogo) a materiais existentes:

Clique em RECONCILIAR
O sistema escaneia itens que nao tem vinculo com o catalogo
Para cada item, voce pode: vincular a um material existente, cadastrar como novo, ou excluir
📈 Relatorio Financeiro F9 ▼
Somente Admin

Relatorios financeiros completos com graficos e comparativos.

Gerar relatorio

Acesse RELATORIO (F9)
Selecione o mes/ano desejado
O sistema gera automaticamente:
  • Painel financeiro — receita, custo, saldo do periodo
  • Grafico mensal — evolucao visual
  • Detalhe por obra — custos por etapa em cada obra
  • Comparativo — mes atual vs anterior

Exportar PDF

Com o relatorio na tela, clique em EXPORTAR PDF
O sistema gera o PDF para download/impressao
💰 Custos CEF (Repasses) F10 ▼
Somente Admin

Controle de repasses da Caixa Economica Federal e custos por obra.

Visualizar custos por obra

Acesse CUSTOS CEF (F10)
As obras aparecem em cards com resumo: lancamentos, adicionais, repasses
Clique em uma obra para ver o detalhe completo com tabela mensal

Registrar repasse CEF

No detalhe da obra, clique em + REPASSE
Preencha: numero da medicao, valor, data do credito, tipo (PLS ou outro)
Clique em SALVAR

Gerar relatorio de custos

No detalhe da obra, clique em RELATORIO
O sistema gera um relatorio completo de custos vs repasses
🗄 Dados (Gestao de Usuarios) F11 ▼
Somente Admin

Gerenciamento de usuarios que acessam o sistema.

Criar novo usuario

Acesse DADOS (F11)
Clique em + NOVO USUARIO
Preencha: nome de usuario (login), senha, nome completo
Selecione o perfil:
  • Admin — acesso total
  • Operacional — lanca NF, estoque, obras, diarias
  • Mestre — somente diarias
Clique em SALVAR

Editar ou desativar usuario

Na lista de usuarios, clique em EDITAR
Altere nome, senha ou perfil conforme necessario
Para desativar (ex: funcionario que saiu), clique em DESATIVAR
Usuarios desativados nao conseguem fazer login, mas o historico de lancamentos deles e mantido.
⚙ Setup F12 ▼
Somente Admin

Configuracoes tecnicas do sistema.

SQL de Setup

Contem o SQL completo para criar todas as tabelas do sistema no Supabase. Usar apenas na primeira instalacao ou quando uma nova tabela for adicionada.

Clique em COPIAR SQL
Abra o Supabase SQL Editor
Cole e execute

Resetar menu

Se a ordem dos menus ficou bagunçada:

Clique em RESETAR MENU
A ordem volta ao padrao (F1 a F12)
O menu lateral pode ser reorganizado arrastando os botoes. O setup reseta isso.
